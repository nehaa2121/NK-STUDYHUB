Introdution to Computers

1.Introduction:

What is Computer?

  • An electronic device or machine.
  • Works under stored instructions (programs).
  • Capable:
    1. Input - Accepting data.
    2. Processing - Manipulating data acording to rules.
    3. Output - Producing information
    4. Storage - Saving data/results for future use.

Computer System Components

  1. Hardware
    • The physical parts of a computer(monitr, keyboard, CPU, etc.).
    • Tangible - you can see and touch them.
  2. Software
    • A set of programs, instructions, and procedures that tell hardware what to do.
    • Includes:
      • System Software - Opearating systems, utilities.
      • Application Software - Programs for specific tasks (e.g., MS Word).

2. Characteristics of Computers:

  1. Speed:
    • Works much faster than humans.
    • Can execute millions on instructons per second.
  2. Accuracy:
    • Very accurate if instructions are correct.
    • Errors only happens if input/instructions are wrong.
  3. Automation:
    • Once started, performs takes automatically without further help.
  4. Versatility:
    • Can do different types of work (calculations, games, music, etc.).
  5. Diligence:
    • Never gets tired or bored.
    • Can repeat taks continuously with the same efficiency.
  6. Efficiency:
    • Works on repeated tasks without losing speed or quality.
  7. Storage Capability:
    • Stores huge amounts of data in a small space (hard disks, CDs, pen drives).

3. Basic Block Diagram of Computer

A computer works through Input → Processing → Storage → Output.
It has four main units.:

1. Input Unit

  • Accepts data and instructions.
  • Converts them into a form the computer can process.
  • Examples:
    • Keyboard
    • Mouse
    • Scanner
    • Microphone
    • Digital camera

2. Processing Unit (CPU) - The "Brain" of the Computer

  1. Control Unit (CU)
    • Directs all operations inside the computer.
    • Manages the flow of data between units.
  2. Arithmetic & Logic Unit (ALU)
    • Performs calculations (addition, subtraction, etc.).

    • Handles logical decisions (comparing values, conditions).
  3. Storage/Memory Unit
    • Primary Memory (RAM):
      • Temporarily stores data and instructions while the computer is working.
      • Loses data when power is off (volatile).
    • Secondary Storage:
      • Permanent storage for data and programs.
      • Examples: Hard drive, SSD, CD/DVD, Pen drive, Memory card.

3. Output Unit

  • Shows the processed information to the user.
  • Examples:
    • Monitor
    • Printer
    • Speakers
    • Projector

Storage Devices Examples

  • Magnetic: Hard disk, Floppy disk, Zip drive.
  • Optical: Pen drive, Memory card, Micro drive.
  • Other: Tape drives, External hard drives.

Key Points/Tips

  • CPU = CU + ALU + Memory Unit
  • RAM = Temporary (volatile) memory.
  • Secondary Storage = Permanent memory.
  • IPOS cycle: Input → Processing → Output → Storage.
  • Hardware = Physical parts;
    Software = Programs that control hardware.

4. Hardware, Software and types of software and data

1. Computer Hardware

  • Computer Hardware is the physical machinery of a computer.
  • Think of it as all the parts you can touch, like the monitor, keyboard, mouse, or the internal chips like ICs (integrated Circuits).

2. Computer Software

  • Computer Software is a set of programs or instructions that tells the hardware what to do.
  • It's the non-physical part of the computer.
  • There are two main types:
    • Application Software: Designed for a specific purpose, like MS Word for document or MS Excel Word for documents or MS Excel for spreadsheets.
    • System Software: Manages and controls the computer's hardware. It allows application software to run. It includes:
      • BIOS/Firmware: A program on a chip that starts the computer. It performs a Power On Self Test (POST) to check the hardware and then initiates the booting process, which loads the operating system.
      • Operating System (OS): The main program that manages all computer resources and acts as an interpreter between you and the hardware. Examples are Windows and Linux.

3.Computer data

  • Computer Data is any information stored or processed by a computer.
  • This includes documents, images, audio, and software
  • Data Processing is the process where the computer's CPU takes raw input data and converts it into a useful output.
