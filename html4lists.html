HTML List

In HTML, lists are used to group sets of related items.

There are 3 types of lists in HTML


1. Ordered List (<ol>)

  • Displayed items in numbered order
  • Used when sequence matters (eg :- Steps, instructions)

Syntax:-

<ol>
    <li>Wake Up</li>
    <li>Brush teeth</li>
    <li>Study</li>
</ol>

Output:

            
    
                
  1. Wake Up
    2. 
                
  2. Brush teeth
    3. 
                
  3. Study
    4.

Attributes

  • type="A" :- A, B, C ....
  • type = "a" :- a, b, c ....
  • type= "I" :- I, II, III, ....
  • start = "5" :- starts numbering from 5

Example:

            <type = "A" start = "3">

Unordered List (<ul>)

  • Displayed items with bullet points
  • Used when order doesn't matter

Syntax:

 <ul>
      <li>Pen</li>
      <li>Notebook</li>
      <li>Bag</li>
 </ul>

Output:

                
    
                    
  • Pen
    • 
                    
  • Notebook
    • 
                    
  • Bag
    •
