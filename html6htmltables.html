What is a Table in HTML

A table is used to display data in rows and columns format - just like in Excel

We use the <table> tag to create a table.

Basic Table Tags:

Tag Meanings
<table> Defines the table
<tr> Defines a table row
<td> Defines a table data cell
<th> Defines a table head or cell (bold + centered by default)

Example: Basic Table

<!DOCTYPE html>
<html>
<head>
   <title>Basic able Example</title>
</head>
<body>
    <h2>Student Details</h2>
    <table border = "1">
        <tr>
          <th>Name</th>
          <th>Course</th>
          <th>Marks</th>
        </tr>
        <tr>
          <td>Riya</td>
          <td>BBA</td>
          <td>85</td>
        </tr>
        <tr>
          <td>Neha</td>
          <td>BCA</td>
          <td>80</td>
        </tr>
    </table>
</body>
</html>

Output:

Student Detail

Name Course Marks
Riya BBA 85
Neha BCA 80

Explanation

  • <table border = "1"> :- Adds a simple border to the table.
  • <tr> :- Creates a row
  • <th> :- Creates a header cell (bold and centered).
  • <td> :- Creates normal data cells

Colspan and Rowspan

Colspan:

Used to merge multiple columns into one

<table border = "1">
  <tr>
    <th colspan = "3">Student Information</th>
  </tr>
  <tr>
    <th>Name</th>   
    <th>Course</th>   
    <th>Marks</th>   
  </tr>
  <tr>
    <td>Neha</td>
    <td>BCA</td>
    <td>80</td>
  </tr>
</table>

Output:

Student Information Information
Name Courses Marks
Neha BCA 80

Explanation:

  • colspan = "3" merges 3 columns into one header cell :- "Student Information" spans across all columns.

Rowspan:

Used to merge multiple rows into one.

<table border = "1">
  <tr>
    <th rowspan = "2">Name</th>
    <td>Neha</td>
  </tr>
  <tr>
    <td>Riya</td>
  </tr>
</table>
