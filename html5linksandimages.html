HTML LINKS AND IMAGES

1. Anchor tag (<a>)

used to create hyperlinks (links to other pages, websites, or files)

Syntax:

<a href = "https://www.google.com">Visit Google</a>

2. Open Link in New Tab

To make the link open in a new browser tab, use the attribute target = "_blank"

Example:

<a href = "https://www.google.com" target = "_blank">Open Google in New Tab

3. Image Tag (<img>)

Used to display an image on a webpage.

Syntax:

<img src = "img.jpg" alt = "Description of image">
  • src - path or URL of the image
  • alt - text shown if image is not found (also used for accesibility)

Example:

<img src = "dog.jpg" alt = "A cute black dog">

4. Linking an image (clickable Image)

You can make an image clickable by placing it inside an <a> tag.

Example:

<a href = "https://www.google.com" target = "_blank"> <img src = "google-logo.png" alt = Google Logo" width = "150">
